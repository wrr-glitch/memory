# 바이브 코딩 강의 랜딩 페이지 개발 계획서

교사 크리에이터들을 위한 '바이브 코딩' 강의를 홍보하는 트렌디하고 화려한 랜딩 페이지를 제작합니다. 사용자가 요청한 Next.js/Tailwind 스타일의 고퀄리티 UI를 구현하며, 부드러운 애니메이션을 통해 '바이브'를 극대화합니다.

## 프로젝트 개요
- **목표**: 교사들의 창작 본능을 깨우는 '바이브 코딩' 강의 홍보 및 수강 유도
- **디자인 컨셉**: 
  - **Trendy & Flashy**: 네온 그라데이션 (Purple to Pink), 깊이감 있는 다크 배경
  - **Interactive**: 스크롤에 반응하는 요소 등장 애니메이션, 마우스 호버 효과
- **기술 스택**: 
  - **Styling**: Tailwind CSS (CDN 활용하여 설치 없이 즉시 실행 가능)
  - **Animations**: AOS (Animate On Scroll) 라이브러리 + CSS Keyframes
  - **Structure**: Semantic HTML5 (Next.js 컴포넌트 스타일로 구조화)

## 주요 섹션 구성
1. **Hero Section**: "코딩에도 바이브가 필요하다" - 강력한 헤드라인과 애니메이션 배경
2. **About Section**: 왜 교사 크리에이터에게 바이브 코딩인가? (감각적인 카드 뉴스 스타일)
3. **Curriculum Section**: 단계별 강의 내용 (화려한 네온 보더 적용)
4. **Teacher's Review**: 먼저 경험한 선생님들의 후기 (슬라이드 또는 그리드)
5. **CTA Section**: 지금 바로 바이브에 합류하기 (빛나는 버튼 효과)

## 사용자 검토 및 동의 필요 사항
> [!IMPORTANT]
> **기술 스택 관련 안내**: 현재 환경에서 Next.js 빌드 도구(Node.js/NPM) 사용에 제한이 확인되었습니다. 따라서 **Next.js와 동일한 결과물을 내면서도 별도의 설치 과정 없이 즉시 `index.html` 파일을 열어 확인하고 배포할 수 있는 'Tailwind CSS 기반 고성능 단일 페이지'**로 제작해 드리고자 합니다. 
> 
> 이 방식은 초보자 프로젝트 관리 측면에서 훨씬 가볍고 빠르며, 결과물의 시각적 완성도는 Next.js로 만든 것과 차이가 없도록 최선을 다하겠습니다. 이렇게 진행해도 될까요?

## 단계별 실행 계획

### 1단계: 프로젝트 폴더 및 기본 구조 설정
- `vibe-coding-landing` 폴더 생성 및 기본 파일 구성
- Tailwind CSS 및 AOS 애니메이션 라이브러리 연결

### 2단계: 핵심 디자인 시스템 구축
- Tailwind 설정을 통한 커스텀 네온 컬러 및 그라데이션 정의
- 글로벌 애니메이션 키프레임 (Floating, Glow) 작성

### 3단계: 섹션별 UI 구현
- Hero 섹션부터 CTA 섹션까지 순차적으로 화려하게 디자인
- 각 섹션에 스크롤 애니메이션 (`data-aos`) 적용

### 4단계: 반응형 및 최종 폴리싱
- 모바일 화면 최적화 (모바일에서도 화려함 유지)
- 스무스 스크롤 로직 및 인터랙션 디테일 강화

## 검증 계획
- **애니메이션 확인**: 스크롤 시 요소들이 부드럽게 나타나는지 검토
- **디자인 검수**: 다크 모드에서의 시독성과 네온 컬러의 조화 확인
